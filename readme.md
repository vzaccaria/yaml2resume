## Install

Install it with

```
npm install yaml2resume
```
## Usage

```
Usage:
    yaml2resume [ -t TEMPLATE ] [ -l ]
    yaml2resume list
    yaml2resume ( -h | --help )

Options:
    -h, --help                  help for yaml2resume
    -t, --template TEMPLATE     template file to use     ( 'latex-jr' )
    -l, --latex                 compile the output with `xelatex`.

Commands:
    list                        list current templates

Arguments:

```
